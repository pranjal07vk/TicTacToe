Tic Tac Toe 

Description: 
A fully interactive Tic Tac Toe game built using Vanilla JavaScript, HTML, and CSS. The game dynamically updates the DOM based on user interactions and manages state on the client side.

Problem Statement:
Create a browser-based game that demonstrates event-driven programming, DOM manipulation, and frontend logic without using any frameworks.

Features:
- Dynamic game board generation
- Real-time UI updates
- Turn-based play
- Win & draw detection
- Restart game option

DOM Concepts Used:
- createElement()
- addEventListener()
- dataset attributes
- Dynamic rendering
- Conditional DOM updates

Tech Stack:
- HTML5
- CSS3
- Vanilla JavaScript

How to Run:
1. Clone the repo
2. Open `index.html` in your browser
3. Play the game!

Limitations
- No AI opponent (2-player only)
- No online multiplayer
